# **App Name**: Static Gallery

## Core Features:

- Secure Login: Hardcoded admin credentials for accessing the upload feature and the gallery.
- Photo Upload: Admin-only feature to upload new images directly to the UI project (public directory).
- Video Upload: Admin-only feature to upload new video files directly to the UI project (public directory).
- Document Upload: Admin-only feature to upload new PDF documents directly to the UI project (public directory).
- Photo Display: Display all images in a gallery-like view on the 'Photos' tab.
- Video Display: Display all videos in a gallery-like view on the 'Videos' tab.
- Document Display: Display all documents in a list view on the 'Documents' tab.
- Tab Navigation: Simple tab-based navigation to switch between 'Photos', 'Videos', and 'Documents'.
- Admin Delete: Admin-only feature to delete photos, videos, and documents from the UI project.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to suggest professionalism.
- Background color: Light gray (#F0F2F5), a very desaturated tint of the primary color, for a clean backdrop.
- Accent color: Violet (#9575CD) to bring more vibrancy.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral aesthetic.
- Simple, outline-style icons for gallery actions like upload and download.
- Grid-based layout for the photo and video galleries.
- Mobile-responsive design to ensure optimal viewing experience across different screen sizes.
- Subtle fade-in animations when loading new images or videos.
